# HETIC Terraform project

## 🚀 Install and Launch Terraform 🚀
To install :
```bash
$ brew tap hashicorp/tap
```

```bash
$ brew install hashicorp/tap/terraform
```
To launch :
```bash
$ terraform apply
```
